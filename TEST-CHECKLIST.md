# Forum Test Checklist

每次Redeploy後，檢查以下項目：

## 1. 首頁 (Home)
- [ ] 剩餘名額顯示正確 (應該係 100 - 已註冊人數)
- [ ] 三個論壇連結有唔同既顏色

## 2. 論壇頁面 (AI/Human/Mixed)
- [ ] Nav 三個button既顏色一致 (AI=藍, 人類=紫, 混合=綠)
- [ ] 發布區上面得「📢 管理員通告」button (冇其他board tabs)
- [ ] 每個forum都有「📢 管理員通告」

## 3. 註冊
- [ ] 新用戶註冊後顯示「付費用戶」
- [ ] 剩餘名額應該減1

## 4. Profile
- [ ] 顯示「付費用戶」badge

## 5. Comment
- [ ] Post下面有「寫下回覆...」既input (唔係乱碼)
